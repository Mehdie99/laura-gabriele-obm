# Site Laura Gabriele · OBM

Landing page personnelle de **Laura Gabriele**, Online Business Manager.
Positionnement : *« Je structure et je pilote votre opérationnel à votre place, pour que vous puissiez vous concentrer sur ce qui compte. »*

Le site est un **fichier HTML autonome** (`index.html`) : aucune dépendance, aucun build. Il s'ouvre directement dans un navigateur et se déploie tel quel.

---

## ✍️ Zones à personnaliser

Tout le contenu est rédigé. Les éléments à remplacer par Laura sont **surlignés en terracotta** dans la page, marqués `[À PERSONNALISER : ...]`. À compléter :

- **Photo** : remplacer le placeholder « LG » (section *Qui je suis*) par une vraie photo
- **Parcours** : le paragraphe de présentation de Laura
- **Chiffres** : barre de preuve (années d'expérience, business structurés, heures rendues)
- **Prix** : les 3 offres (Audit / Sprint Structuration / Pilotage continu) + la value stack
- **Témoignages** : 4 témoignages (exemples crédibles en place)
- **Liens** : lien de réservation d'appel + profil LinkedIn (footer)
- **Webhook du diagnostic** : variable `WEBHOOK_URL` dans le `<script>` (pour recevoir les résultats du quiz)

---

## 🖥️ Modifier le site

Tout se trouve dans `index.html`. Ouvrir le fichier dans un éditeur (VS Code), modifier le texte, enregistrer, recharger le navigateur.

## 🚀 Déployer

Le plus simple, gratuit :

1. Créer un compte sur [vercel.com](https://vercel.com) (ou Netlify)
2. Importer ce dépôt GitHub
3. Déploiement automatique à chaque modification poussée sur la branche `main`

Aucune configuration nécessaire (site statique).
